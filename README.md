# Task Manager API

Aplicação simples de gestão de tarefas desenvolvida com FastAPI.
Este projeto foi criado como parte do meu aprendizado em APIs REST, incluindo conceitos como CRUD, validação com Pydantic e rotas estruturadas.

# Funcionalidades

-Criar tarefas
-Listar todas as tarefas
-Buscar tarefa por ID
-Atualizar tarefa
-Excluir tarefa
-ID único automático com UUID

# Tecnologias utilizadas

-Python
-FastAPI
-Pydantic
-Uvicorn
-UUID
